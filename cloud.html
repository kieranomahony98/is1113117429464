Software as a Service(SaaS) is a cloud computing service which allows users to devolp,run and manage projects using a website without the need of installing an application. It elimates the cost of hardware aquisation aswell as maintaince and aswell as other costs such as licensing. It also allows for greater scalabilty as the organisation will not need to purchase external storage.







Platform as a Service(PaaS) typically does not replace the IT infrastructure of a business.Instead its utilised to provide services such as application hosting or java development. as a results users can focus on creating and running applications rather then maintaining infrastructure and services. It will also allow teams to collaborate and work toghether despite being in different geographical regions.
